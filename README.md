# Bank-Churn-Segmentation-
This project focuses on predicting customer churn in the banking sector using a Random Forest classification model. Customer churn, which refers to customers leaving a bank for competitors, directly impacts revenue and long-term profitability
Objectives
	•	Identify key factors influencing customer churn
	•	Build and optimize a Random Forest classification model
	•	Evaluate performance using multiple metrics
	•	Visualize churn patterns and feature importance
	•	Simulate real-world churn prediction using user input
  Dataset
	•	Source: Kaggle – Churn Modelling Dataset
	•	Total Records: 10,000 customers
	•	Target Variable:
	•	Exited (0 = Stayed, 1 = Churned)
  Model Development
	•	Algorithm: Random Forest Classifier
	•	Splitting Criterion: Entropy (Information Gain)
	•	Cross-Validation: 5-Fold Cross Validation
